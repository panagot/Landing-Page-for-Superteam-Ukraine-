# Figma handoff — Superteam Ukraine landing page

This folder contains everything needed to **build the required Figma file** for submission (desktop + mobile, component library, style guide) or to hand the design off to developers.

**Submission requirement:** A working Figma file with:
1. **Landing page** (desktop + mobile)
2. **Component library**
3. **Style guide**

Use the documents here to recreate the design in Figma so it matches the live site. The live implementation is in the parent repo (`superteam-ukraine-landing`).

---

## Contents

| File | Purpose |
|------|--------|
| **STYLE-GUIDE.md** | Colors (hex), typography, spacing grid, shadows, borders. Use for the Style guide page in Figma. |
| **COMPONENTS.md** | List of UI components and variants (buttons, cards, header, footer, etc.). Use to build the Component library in Figma. |
| **PAGE-STRUCTURE.md** | Section order, layout, and copy for desktop and mobile. Use for the Landing page frames. |

---

## Quick reference

- **Primary theme:** Dark (dark backgrounds, light text, yellow/gold accent).
- **Fonts:** Plus Jakarta Sans (body), Rajdhani (headings). Google Fonts.
- **Max content width:** 1152px (72rem). Horizontal padding 16px (mobile), 24px (desktop).
- **Section padding:** 80px vertical (mobile 20), 96px (desktop 24).

Create a new Figma file, add three main pages: **Style guide**, **Components**, **Landing (Desktop)**, **Landing (Mobile)**. Populate them from these specs.
